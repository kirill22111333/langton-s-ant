# langton's ant
## Пример использования
```php
$game = new game;
$game->act = str_split('RL'); // Создание паттерна поведения. Должен быть массивом!
$game->generate(3); // Создание карты. Карта будет подстраиваться под муравья.
$game->put_ant([1, 1]); // Создание муравья, в параметрах можно указать положение.

// Запуск бесконечного цикла
while(true){
    $game->next_step(); // Следующий шаг.
    echo $game->compile()."\n"; // Вывод.
}
```
